Diabetic Retinopathy Detection

Do you have Diabetic Retinopathy?

Problem Statement

The purpose of this task is to classify whether a patient has Diabetic Retinopathy or not based on his/her eye images.

Dataset

We use the dataset available on Kaggle:Kaggle

Data Analysis and Machine learning

In this project, we will dive into the images, caption and tags . We will implement ETL (Extract, Transform and Load) pipeline, ML pipeline and text classification to classify whether an image contains Pav-Bhaji or not. We need to create proper feature vectors as the text contains Hindi, English and emojis
