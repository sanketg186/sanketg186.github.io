Anime Image Generation

Can we generate new anime images?

Problem Statement

The purpose of this task is to explore various image generation deep learning models that learn the latent representation of the image data and later new images can be sampled from the model.

Dataset

We use the dataset availabe on Kaggle.

Deep learning

In this project, we will be implementing vanilla VAE, Hierarchical and DCGAN to generate anime images.
